<프로젝트 파일 목록>
-config : 언리얼의 에디터, 엔진 등의 설정들이 들어간 폴더
-contents : 에셋(모델링)이 들어간 파일
-프로젝트 폴더 : 실행하면 바로 언리얼 열어짐
--이 세가지가 가장 중요한 폴더임

<에디터 팁>
-num rock 키고 end 키 -> 오브젝트가 바닥에 고정됨

-만든 모델링을 저장하려면 우측 창 디테일 -> 고급 -> 스태틱 매시 생성 버튼 클릭

-창이 바뀌거나 닫혔을 경우 상단 바 창 - 레이아웃 불러오기 - 디폴트 레이아웃 클릭

-언리얼 시작 시 원하는 맵으로 시작하고 싶은 경우 -> 상단 바 편집 - 프로젝트 - 맵&모드 - 에디터 시작 맵과 게임 기본 맵(말 그대로 첫 실행 시 시작 화면)을 원하는 맵으로 지정

-만약 엔진이 버그가 뜬다면? -> 에픽게임즈 런처 - 라이브러리에서 엔진 버전 - 검증 버튼 클릭

-머테리얼 : 생성, 표 창에서 1을 누르고 좌클릭하면 패널이 새로 추가됨(1~4까지) -> 지정 후 적용 하면 맵 내 적용 물체가 머테리얼 반영됨
https://dev.epicgames.com/documentation/ko-kr/unreal-engine/physically-based-materials-in-unreal-engine - 머테리얼 상세 팁
--+이니시브 컬러 : 빛 효과 적용(빨강색 기반으로 만들면 태양 만들 수 있음)

-텍스처는 그림판 등으로 만들어서 언리얼 콘텐츠 폴더에 넣을 수 있음
--텍스처는 머테리얼에 적용가능
--1로 그린 부분 색 지정 가능, 1-로 그 반대 부분 색 지정 가능
--좌측 바의 블렌드 모드로 masked(그린 부분 투명화), translucent(투명 그라데이션) 등을 설정 가능.
--선을 alt 키 눌러서 삭제 가능

-플레이 상태에서 F8 버튼을 누르면 커서 및 화면 조작 사용 가능함

<그 외 팁>
-유튜브에서 '언리얼을더쉽게'로 교육 가능
-톨토이스 깃 설치 : 64윈도우버전 설치한 후 패드워드입력 창에서 하단 링크 클릭해 깃 다운(쭉 넥스트 눌러서 설치) -> 깃이 설치되면 톨토이스 패스워드 입력 창에서 자동으로 경로가 지정됨 -> 아이디랑 이메일 입력 후 마침 -> 해당 언리얼 프로젝트 폴더에서 우클릭으로 깃이 있는지 확인(추가 옵션 표시로 나머지가 표시되는 지 확인)
-깃 연결 방법 : 깃허브에서 로그인 후 방을 만든다. -> 해당 언리얼 메인 폴더에서 우클릭 - 깃 클론 클릭 -> 확인
-깃 사용 방법 : 원하는 파일 우클릭 -> 톨토이스 깃 -> add 클릭 -> 빈 칸에 우클릭 후 commit 클릭 -> 메시지 입력 후 커밋(회사에서는 사유 목적 기술해야 함) -> 상단 바 보기 - 표시 - 숨긴 항목 클릭하면 '.git'이 보임 -->> 빈 칸 우클릭 - 톨토이스 - 푸시 버튼 클릭하여 깃에 전송(도중에 로그인 화면 나오면 체크) -> 해당 깃 방을 새로고침하면 전송한 파일이 뜸
--그 반대 방법도 깃 클론을 이용해 사용해 볼 수 있다.(단 로그인 필요)
--수정 후 반영도 쉽다.(커밋 -> 푸쉬 바로 클릭) 
--지우지 않으면서 집과 회사의 자료를 공유하는 방법 -> 풀(pull)을 사용하면 된다.
--굳이 USB를 사용하지 않아도 폴더에서 자유로히 자료를 갱신할 수 있다!
-지우는 것도 파일은 그냥 지우되 커밋 과정을 수행해야 한다.
